# 📊 Social Media Impact on Students – EDA Project

## 📌 Project Overview
This project performs **Exploratory Data Analysis (EDA)** on a dataset that examines the impact of social media usage on students' **mental health, sleep patterns, and academic performance**.

---

## 🎯 Objectives
- Understand dataset structure and features  
- Perform data cleaning and preprocessing  
- Analyze patterns and relationships  
- Derive actionable insights  

---

## 📂 Project Structure

├── data.csv
├── data_preprocessing.ipynb
├── EDA.ipynb
├── Student Social Media analysis.docx
├── requirements.txt
└── README.md


---

## 🧠 Key Analysis

### 🔹 Dataset Understanding
- Data types and feature classification  
- Missing values and duplicates  

### 🔹 Data Cleaning
- Handling null values  
- Fixing inconsistent data  
- Validating ranges  

### 🔹 Exploratory Data Analysis
- Distribution of usage hours  
- Mental health score trends  
- Sleep patterns  
- Platform usage  

### 🔹 Relationship Analysis
- Usage vs Mental Health  
- Usage vs Sleep  
- Sleep vs Mental Health  
- Academic impact analysis  

### 🔹 Segmentation
- Low usage (<2 hours)  
- Medium usage (2–5 hours)  
- High usage (>5 hours)

## 🚀 Usage
- Run data_preprocessing.ipynb for cleaning
 - Run EDA.ipynb for analysis and visualization
  ---
  
## 📈 Key Insights
- High social media usage correlates with lower mental health scores
- Less sleep (<6 hours) negatively impacts well-being
- Excess usage affects academic performance
